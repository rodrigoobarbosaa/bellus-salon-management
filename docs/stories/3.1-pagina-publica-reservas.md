## Story

Como **cliente**, quero **acessar uma página pública do salão** (sem criar conta), para que possa ver informações do salão e fazer uma reserva online.

---

## Acceptance Criteria

- [ ] AC-01: Rota pública `/booking/[slug]` resolve pelo slug do salão
- [ ] AC-02: Página mostra nome, logo, endereço, horários de funcionamento, WhatsApp e redes sociais do salão
- [ ] AC-03: Página detecta idioma do browser e mostra conteúdo em ES/PT/EN/RU
- [ ] AC-04: Layout responsivo mobile-first com cores do salão (cor_primaria)
- [ ] AC-05: Botão "Reservar" leva ao fluxo de booking
- [ ] AC-06: Página 404 amigável se slug não existe
- [ ] AC-07: Sem autenticação necessária (acesso público, bypassa RLS)
