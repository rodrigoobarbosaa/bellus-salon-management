# Story 3.1 — Página Pública de Reservas

**Epic:** EPIC-03 Portal Público & Notificações
**Status:** Done
**Prioridade:** MUST HAVE
**Depende de:** EPIC-02 (completo)
**PRD Refs:** FR-02.1, FR-02.2

## Story

Como **cliente**, quero **acessar uma página pública do salão** (sem criar conta), para que possa ver informações do salão e fazer uma reserva online.

---

## Acceptance Criteria

- [x] AC-01: Rota pública `/booking/[slug]` resolve pelo slug do salão
- [x] AC-02: Página mostra nome, logo, endereço, horários de funcionamento, WhatsApp e redes sociais do salão
- [x] AC-03: Página detecta idioma do browser e mostra conteúdo em ES/PT/EN/RU
- [x] AC-04: Layout responsivo mobile-first com cores do salão (cor_primaria)
- [x] AC-05: Botão "Reservar" leva ao fluxo de booking
- [x] AC-06: Página 404 amigável se slug não existe
- [x] AC-07: Sem autenticação necessária (acesso público, bypassa RLS)
