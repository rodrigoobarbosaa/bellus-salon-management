## User Story

**Como** proprietário do salão,
**Quero** ter um dashboard com navegação clara e responsiva,
**Para que** eu consiga acessar todas as funcionalidades do Bellus de forma intuitiva.

## Contexto

Layout principal da aplicação. Sidebar com navegação para todos os módulos (Agenda, Clientes, Caixa, Fiscal, Marketing, Configurações). Deve ser mobile-first e responsivo.

## Acceptance Criteria

- [ ] AC-01: Sidebar com links para: Início, Agenda, Clientes, Caixa, Fiscal, Marketing, Configurações
- [ ] AC-02: Header com: logo/nome do salão, seletor de idioma, menu do usuário (logout)
- [ ] AC-03: Layout responsivo: sidebar colapsada em mobile, expandida em desktop
- [ ] AC-04: Rota ativa destacada na sidebar
- [ ] AC-05: Página inicial do dashboard mostra mensagem de boas-vindas com nome do salão
- [ ] AC-06: Loading skeleton enquanto dados carregam
- [ ] AC-07: Carregamento inicial em menos de 2 segundos (LCP)
- [ ] AC-08: Cor primária do salão aplicada nos elementos de destaque (sidebar, botões primários)
