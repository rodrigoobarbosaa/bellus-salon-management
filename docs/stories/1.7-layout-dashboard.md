## User Story

**Como** proprietário do salão,
**Quero** ter um dashboard com navegação clara e responsiva,
**Para que** eu consiga acessar todas as funcionalidades do Bellus de forma intuitiva.

## Contexto

Layout principal da aplicação. Sidebar com navegação para todos os módulos (Agenda, Clientes, Caixa, Fiscal, Marketing, Configurações). Deve ser mobile-first e responsivo.

## Acceptance Criteria

- [x] AC-01: Sidebar com links para: Início, Agenda, Clientes, Caixa, Fiscal, Marketing, Configurações
- [x] AC-02: Header com: logo/nome do salão, seletor de idioma, menu do usuário (logout)
- [x] AC-03: Layout responsivo: sidebar colapsada em mobile, expandida em desktop
- [x] AC-04: Rota ativa destacada na sidebar
- [x] AC-05: Página inicial do dashboard mostra mensagem de boas-vindas com nome do salão
- [x] AC-06: Loading skeleton enquanto dados carregam
- [x] AC-07: Carregamento inicial em menos de 2 segundos (LCP)
- [x] AC-08: Cor primária do salão aplicada nos elementos de destaque (sidebar, botões primários)
