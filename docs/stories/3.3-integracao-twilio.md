# Story 3.3 — Integração Twilio WhatsApp/SMS

**Epic:** EPIC-03 Portal Público & Notificações
**Status:** Ready for Dev
**Prioridade:** MUST HAVE
**Depende de:** Story 3.2
**PRD Refs:** FR-05.1, FR-05.2

---

## Story

Como **proprietário**, quero que o sistema **envie notificações WhatsApp/SMS** via Twilio quando um agendamento é criado, para que clientes recebam confirmação automática.

---

## Acceptance Criteria

- [ ] AC-01: Integração Twilio configurada com env vars (TWILIO_ACCOUNT_SID, TWILIO_AUTH_TOKEN, TWILIO_WHATSAPP_FROM, TWILIO_SMS_FROM)
- [ ] AC-02: Função `sendWhatsApp(to, message)` envia mensagem via Twilio WhatsApp API
- [ ] AC-03: Função `sendSMS(to, message)` como fallback se WhatsApp falhar
- [ ] AC-04: Notificação registrada em `notificacoes_log` com status (pendente → enviado/falhou)
- [ ] AC-05: Erro de envio não bloqueia criação do agendamento
- [ ] AC-06: Env vars opcionais — se não configuradas, notificações são apenas logadas

---

## Dev Notes

- Package: `twilio` npm
- Twilio WhatsApp sandbox para dev: `whatsapp:+14155238886`
- Fallback chain: WhatsApp → SMS → log only
- Todas as chamadas Twilio com try/catch — nunca bloquear o fluxo principal
- Service client para atualizar notificacoes_log
