### ❌ Issues Encontrados

#### [HIGH-1] `.gitignore` ignora `.env.example`

**Arquivo:** `bellus/.gitignore`
**Problema:** O pattern `.env*` cobre `.env.example`, impedindo que o template seja commitado ao repositório. Novos desenvolvedores não terão o arquivo de referência.
**Fix:** Alterar para `.env*.local` (ignora `.env.local`, `.env.production.local`, etc.) mantendo `.env.example` rastreável pelo git.

---

#### [HIGH-2] Root `middleware.ts` ausente

**Arquivo:** Faltando `bellus/src/middleware.ts`
**Problema:** A função `updateSession` foi criada em `src/lib/supabase/middleware.ts` mas nunca é chamada pelo Next.js. O middleware Supabase SSR exige um arquivo `src/middleware.ts` que invoque `updateSession` para refresh automático de sessão. Sem isso, tokens de auth expiram silenciosamente.
**Evidência:** Apenas `src/lib/supabase/middleware.ts` existe — sem `src/middleware.ts`.
**Fix:** Criar `bellus/src/middleware.ts` chamando `updateSession` com matcher de rotas protegidas.

---

#### [HIGH-3] `lang="es"` hardcoded no root layout

**Arquivo:** `bellus/src/app/layout.tsx:22`
**Problema:** O atributo `lang` está fixo em `"es"` (Espanhol). O Bellus é um sistema brasileiro; o default deve ser `"pt-BR"`. Impacta acessibilidade (screen readers), SEO e tradução automática do browser.
**Fix:** Alterar `lang="es"` para `lang="pt-BR"` (Story 1.5 tornará dinâmico por rota).

---

#### [MEDIUM-1] `src/hooks/` e `src/stores/` sem `.gitkeep`

**Problema:** Diretórios completamente vazios — git não rastreia diretórios vazios. Futuras stories podem ter confusão ao depender dessas pastas.
**Fix:** Adicionar `.gitkeep` em `src/hooks/` e `src/stores/`.

---

#### [MEDIUM-2] Checkboxes de locale files inconsistentes com Task 7

**Problema:** Task 7 marca `[x]` para criação dos arquivos `pt.json`, `es.json`, `en.json`, `ru.json`, mas apenas `.gitkeep` existe. Conteúdo real é escopo da Story 1.5 — checkboxes devem refletir isso.
**Fix (@dev):** Ajustar checkboxes da Task 7 para indicar que apenas o diretório foi criado (arquivos → Story 1.5).
