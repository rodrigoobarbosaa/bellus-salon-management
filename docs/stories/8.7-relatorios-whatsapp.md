# Story 8.7 — Relatorios de Marketing via WhatsApp

**Epic:** EPIC-08 Marketing IA
**Status:** Ready for Dev
**Prioridade:** MUST HAVE
**PRD Refs:** FR-12.6

---

## Story

Como **proprietario**, quero **receber relatorios semanais no WhatsApp** para acompanhar o marketing sem precisar abrir o sistema.

---

## Acceptance Criteria

- AC1: Relatorio semanal automatico (toda segunda-feira)
- AC2: Conteudo: resumo da semana (agendamentos, receita, campanhas, custo)
- AC3: Alerta em tempo real: campanha esgotou orcamento, avaliacao negativa
- AC4: Comandos por WhatsApp: "pause campanha", "ver relatorio"
- AC5: Usa infraestrutura Twilio existente (FR-05)

---

## Tasks

- [ ] T1: Cron job /api/cron/marketing-weekly-report
- [ ] T2: Gerar texto do relatorio semanal com dados agregados
- [ ] T3: Enviar via Twilio WhatsApp (reutilizar send-notification)
- [ ] T4: Alerta real-time para eventos criticos
- [ ] T5: Webhook para comandos recebidos via WhatsApp
- [ ] T6: i18n (pt, es, en, ru)
