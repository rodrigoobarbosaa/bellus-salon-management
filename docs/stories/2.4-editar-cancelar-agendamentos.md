## Story

Como **proprietário ou profissional**, quero **editar ou cancelar agendamentos existentes**, para que possa reagendar clientes e manter a agenda atualizada.

---

- [x] AC-01: Clicar num agendamento no calendário abre modal com detalhes
- [x] AC-02: Proprietário pode editar: data/hora, serviço, profissional, notas

## Dev Notes

### Status transitions
- `pendente` → `confirmado` | `cancelado`
- `confirmado` → `concluido` | `cancelado`
- `concluido` → (terminal)
- `cancelado` → (terminal)

### Drag-and-drop
FullCalendar `@fullcalendar/interaction` plugin permite:
- `eventDrop` — mover evento para outro dia/hora
- `eventResize` — alterar duração

Ambos devem revalidar conflito antes de salvar.

### Padrões
- Server Actions: `updateAgendamento`, `cancelAgendamento`
- Reutilizar componente de formulário da Story 2.3 (modo edição)
- Confirmação de cancelamento com Dialog
