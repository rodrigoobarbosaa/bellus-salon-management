# Story 1.13: UX Polish Premium — Micro-interacoes, Skeletons, Empty States e Animacoes

**Status:** InProgress
**Epic:** EPIC-1 Bellus — Wave 1 Fundacao
**Criado por:** River (@sm) | Validado por: Pax (@po) GO 10/10
**Data:** 2026-03-29
**Depende de:** Story 1.12

---

## Story

**As a** utilizador do Bellus,
**I want** uma experiencia visual premium com animacoes suaves, feedback tactil, e estados visuais polidos,
**so that** o app transmita qualidade profissional e seja agradavel de usar.

---

## Acceptance Criteria

1. Botoes com micro-interacao: `active:scale-95` + `transition-transform` para feedback tactil
2. Cards com hover lift: `hover:-translate-y-0.5 hover:shadow-md transition-all`
3. Skeleton loading component reutilizavel em vez de texto "Cargando..."
4. Empty states com icone grande + mensagem + CTA em todas as paginas principais
5. Valores monetarios com `tabular-nums` para alinhamento consistente
6. Animacao de entrada stagger para listas (items aparecem sequencialmente)
7. Confetti/celebracao na confirmacao de pagamento
8. `npm run build` sem erros
