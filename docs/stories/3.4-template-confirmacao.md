## Story

Como **proprietário**, quero que as **mensagens de confirmação usem templates com variáveis dinâmicas** e no idioma do cliente, para que a comunicação seja personalizada.

---

## Acceptance Criteria

- [x] AC-01: Template padrão com variáveis: {nome_cliente}, {servico}, {profissional}, {data}, {hora}, {salao}, {endereco}
- [x] AC-02: Template renderizado no idioma_preferido do cliente
- [x] AC-03: Templates default em 4 idiomas (ES, PT, EN, RU)
- [x] AC-04: Mensagem enviada em menos de 30 segundos após criação do agendamento

---

## Dev Notes

- Templates como strings com placeholders `{variavel}`
- Função `renderTemplate(template, variables, locale)` resolve variáveis
- Templates default hardcoded inicialmente (Story 3.6 permite editar)
- Idioma do cliente vem de `clientes.idioma_preferido`
