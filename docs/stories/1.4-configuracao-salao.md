## User Story

**Como** proprietário do salão,
**Quero** configurar as informações do meu salão (nome, logo, endereço, WhatsApp),
**Para que** o sistema seja personalizado com a identidade do meu negócio.

## Contexto

Primeira funcionalidade que o proprietário usa após login. Os dados configurados aqui aparecem no portal público de reservas, nos relatórios e nas notificações WhatsApp. Para o Tati & Rodri Hair Studio, o nome, cor e logo são essenciais para a identidade visual.

## Acceptance Criteria

- [ ] AC-01: Proprietário acessa página de configurações do salão no dashboard
- [ ] AC-02: Formulário com campos: nome, endereço, telefone, WhatsApp Business, horário de funcionamento
- [ ] AC-03: Upload de logótipo (JPG/PNG, máx 2MB) com preview
- [ ] AC-04: Seletor de cor primária do salão (hex picker)
- [ ] AC-05: Links para redes sociais (Instagram URL, Google Maps URL)
- [ ] AC-06: Configuração de moeda (EUR padrão) e fuso horário (Europe/Madrid padrão)
- [ ] AC-07: Dados salvos no Supabase e refletidos imediatamente na UI
- [ ] AC-08: Validação de campos obrigatórios (nome, WhatsApp)
