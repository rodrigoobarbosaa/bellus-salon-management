# Story 8.2 — Interface Conversacional de Marketing

**Epic:** EPIC-08 Marketing IA
**Status:** Ready for Dev
**Prioridade:** MUST HAVE
**PRD Refs:** FR-12.1

---

## Story

Como **proprietario**, quero **conversar em linguagem natural com o sistema** para receber sugestoes de marketing e executar acoes sem conhecimento tecnico.

---

## Acceptance Criteria

- AC1: Chat integrado na pagina /dashboard/marketing
- AC2: Proprietario envia mensagem em linguagem natural (pt/es)
- AC3: Sistema interpreta intencao e propoe acao concreta com orcamento estimado
- AC4: Proprietario pode aprovar acao respondendo "aprova" ou similar
- AC5: Historico de conversas persistido
- AC6: Respostas em menos de 3 segundos (streaming)

---

## Tasks

- [ ] T1: API route /api/marketing/chat com streaming (Claude API)
- [ ] T2: System prompt com contexto do salao (servicos, agenda, clientes, campanhas)
- [ ] T3: Componente ChatPanel com input, mensagens, scroll automatico
- [ ] T4: Streaming de resposta com indicador de digitacao
- [ ] T5: Persistir conversas na tabela marketing_conversas
- [ ] T6: Tool use: propor campanha, consultar metricas, gerar conteudo
- [ ] T7: i18n (pt, es, en, ru)
