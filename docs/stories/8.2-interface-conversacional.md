# Story 8.2 — Interface Conversacional de Marketing

**Epic:** EPIC-08 Marketing IA
**Status:** Done
**Prioridade:** MUST HAVE
**PRD Refs:** FR-12.1

---

## Story

Como **proprietario**, quero **conversar em linguagem natural com o sistema** para receber sugestoes de marketing e executar acoes sem conhecimento tecnico.

---

## Acceptance Criteria

- [x] AC1: Chat integrado na pagina /dashboard/marketing
- [x] AC2: Proprietario envia mensagem em linguagem natural (pt/es)
- [x] AC3: Sistema interpreta intencao e propoe acao concreta com orcamento estimado
- [x] AC4: Proprietario pode aprovar acao respondendo "aprova" ou similar
- [x] AC5: Historico de conversas persistido
- [x] AC6: Respostas em menos de 3 segundos (streaming)

---

## Tasks

- [x] T1: API route /api/marketing/chat com streaming (Claude API)
- [x] T2: System prompt com contexto do salao (servicos, agenda, clientes, campanhas)
- [x] T3: Componente ChatPanel com input, mensagens, scroll automatico
- [x] T4: Streaming de resposta com indicador de digitacao
- [x] T5: Persistir conversas na tabela marketing_conversas
- [ ] T6: Tool use: propor campanha, consultar metricas, gerar conteudo — **DEFERRED** — O chat usa system prompt com instrucoes para sugerir acoes, mas nao implementa tool_use formal da API Claude. O modelo sugere acoes via texto que o usuario pode aprovar manualmente.
- [x] T7: i18n (pt, es, en, ru)
