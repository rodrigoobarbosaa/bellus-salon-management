# Story 9.7: Gerador XML Formato Verifactu (Orden HAC/1177/2024)

**Status:** InProgress
**Epic:** EPIC-09 Verifactu
**Criado por:** River (@sm)
**Data:** 2026-03-30
**Depende de:** Story 9.4 (hash), Story 9.5 (assinatura)
**Complexidade:** L (8 SP)

---

## Story

**As a** sistema Bellus certificado,
**I want** gerar XML no formato exato especificado pela AEAT,
**so that** os registros de facturação possam ser enviados e aceites pelo sistema Verifactu.

---

## Acceptance Criteria

1. Existe módulo `src/lib/verifactu/xml-builder.ts` que gera XML a partir de uma fatura
2. O XML segue o schema da Orden HAC/1177/2024 (encoding UTF-8, namespace correto)
3. Suporta tipo `RegistroAlta` (nova fatura) e `RegistroAnulacion` (anulação)
4. Campos obrigatórios no XML: IDVersion, NIF emisor, NumSerieFactura, FechaExpedicionFactura, TipoFactura, ClaveRegimenIVA, DescripcionOperacion, ImporteTotal, BaseImponible, CuotaRepercutida, TipoImpositivo, Encadenamiento (hash anterior), FirmaMD5 (hash actual)
5. O XML gerado passa validação contra o XSD oficial da AEAT (se disponível)
6. O XML é armazenado no campo `xml_verifactu` da fatura
7. Existe função `buildRegistroAlta(factura)` e `buildRegistroAnulacion(factura)`
8. Testes unitários: XML gerado para fatura normal, fatura retificativa, e primeira fatura (genesis)
9. `npm run build` sem erros
