# Story 7.1 — Widget Agenda do Dia

**Epic:** EPIC-07 Dashboard & KPIs
**Status:** Ready for Dev
**Prioridade:** MUST HAVE
**PRD Refs:** FR-10.1

---

## Story

Como **proprietario**, quero **ver os proximos atendimentos do dia no dashboard** para ter visao rapida da agenda sem navegar para o calendario.

---

## Acceptance Criteria

- AC1: Card mostra lista dos agendamentos de hoje, ordenados por horario
- AC2: Cada item exibe: hora, nome do cliente, servico, profissional
- AC3: Status visual (badge) para cada agendamento: Pendente / Confirmado / Concluido / Cancelado
- AC4: Mostra contagem total de atendimentos do dia
- AC5: Link "Ver agenda completa" redireciona para /agenda
- AC6: Se nao ha agendamentos, exibe empty state amigavel
