# Story 1.5: Sistema i18n — Português, Español, English, Русский

**Status:** Ready
**Epic:** EPIC-1 Bellus — Wave 1 Fundação
**Criado por:** River (@sm)
**Data:** 2026-02-25
**Depende de:** Story 1.1 (projeto configurado), Story 1.4 (layout com seletor de idioma)

---

## Story

**As a** usuário do Bellus (profissional ou cliente),
**I want** poder usar o sistema no meu idioma preferido (PT, ES, EN ou RU),
**so that** a experiência seja natural e confortável, especialmente para a comunidade estrangeira em Valencia.

---

## Acceptance Criteria

1. Sistema suporta 4 idiomas: Português 🇧🇷 (`pt`), Español 🇪🇸 (`es`), English 🇬🇧 (`en`), Русский 🇷🇺 (`ru`)
2. Idioma padrão do sistema: Español (`es`) — região de Valencia
3. Seletor de idioma visível no header do painel — dropdown com bandeiras e nome do idioma
4. Idioma selecionado persiste em cookie e no perfil do usuário (campo `idioma_preferido` em `profissionais`)
5. Idioma do cliente persiste no campo `idioma_preferido` da tabela `clientes`
6. Portal público de reservas detecta idioma do browser automaticamente (`Accept-Language`)
7. Todas as páginas da Wave 1 traduzidas: login, forgot-password, dashboard, menu de navegação
8. Datas exibidas no formato local: `25/02/2026` (PT/ES), `02/25/2026` (EN), `25.02.2026` (RU)
9. Valores monetários exibidos em EUR com separador local: `21,50 €` (PT/ES), `€21.50` (EN), `21,50 €` (RU)
10. Mensagens de erro e feedback de formulários traduzidas em todos os idiomas
11. Notificações WhatsApp/SMS enviadas no idioma do cliente (campo `idioma_preferido`)

## Dev Notes

### Configuração next-intl

```typescript
// src/i18n.ts
import { notFound } from 'next/navigation'
import { getRequestConfig } from 'next-intl/server'

export const locales = ['pt', 'es', 'en', 'ru'] as const
export const defaultLocale = 'es' as const
export type Locale = typeof locales[number]

export default getRequestConfig(async ({ locale }) => {
  if (!locales.includes(locale as Locale)) notFound()

  return {
    messages: (await import(`./locales/${locale}.json`)).default
  }
})
```

### Formatação de Datas e Moeda

```typescript
// src/lib/formatters.ts
export function formatDate(date: Date, locale: string): string {
  return new Intl.DateTimeFormat(locale === 'pt' ? 'pt-BR' : locale, {
    day: '2-digit',
    month: '2-digit',
    year: 'numeric'
  }).format(date)
}

export function formatCurrency(value: number, locale: string): string {
  return new Intl.NumberFormat(locale === 'pt' ? 'pt-BR' : locale, {
    style: 'currency',
    currency: 'EUR'
  }).format(value)
}
// Resultado: formatCurrency(21.50, 'es') → "21,50 €"
//            formatCurrency(21.50, 'en') → "€21.50"
```
