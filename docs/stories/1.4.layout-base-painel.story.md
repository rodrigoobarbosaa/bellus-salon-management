# Story 1.4: Layout Base do Painel de Gestão

**Status:** Ready
**Epic:** EPIC-1 Bellus — Wave 1 Fundação
**Criado por:** River (@sm)
**Data:** 2026-02-25
**Depende de:** Story 1.1, Story 1.3 (auth funcionando)

---

## Story

**As a** profissional do salão Bellus,
**I want** ter um painel de gestão com navegação clara e responsiva,
**so that** eu possa acessar todas as funcionalidades do sistema de forma intuitiva tanto no computador quanto no celular.

---

## Acceptance Criteria

1. Layout do dashboard com sidebar lateral (desktop) e bottom navigation (mobile)
2. Sidebar contém: Logo Bellus, menu de navegação, nome do usuário logado, botão de logout
3. Itens de menu para o **proprietário**: Dashboard, Agenda, Clientes, Serviços, Caixa, Fiscal, Equipe, Configurações
4. Itens de menu para o **profissional**: Dashboard, Minha Agenda, Meus Atendimentos
5. Menu responsivo: sidebar em desktop (≥768px), bottom bar em mobile (<768px)
6. Página de Dashboard inicial com: saudação personalizada, data de hoje, próximos agendamentos (placeholder), faturamento do dia (placeholder)
7. Header do painel com: nome do salão "Bellus", seletor de idioma (PT/ES/EN/RU), avatar do usuário
8. Componente de loading (skeleton) para estados de carregamento
9. Componente de toast/notificação para feedback de ações (sucesso, erro, aviso)
10. Design responsivo validado em: mobile 375px, tablet 768px, desktop 1280px
11. Tema visual: cores dourado (#C9A96E) e preto (#1A1A1A), fonte moderna (Inter)
12. WCAG 2.1 AA: contraste mínimo 4.5:1, navegação por teclado funcional
