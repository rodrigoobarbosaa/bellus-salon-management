# Story 7.2 — KPIs de Faturamento

**Epic:** EPIC-07 Dashboard & KPIs
**Status:** Ready for Dev
**Prioridade:** MUST HAVE
**PRD Refs:** FR-10.2

---

## Story

Como **proprietario**, quero **ver o faturamento em diferentes periodos** para acompanhar a saude financeira do negocio.

---

## Acceptance Criteria

- AC1: Card exibe faturamento de hoje, esta semana, este mes e este trimestre
- AC2: Valores formatados em EUR com locale correto
- AC3: Indicador de variacao percentual vs periodo anterior (ex: +12% vs semana passada)
- AC4: Icone de seta para cima (verde) ou para baixo (vermelho) conforme variacao
- AC5: Dados vem da tabela de transacoes (apenas status=completed)
