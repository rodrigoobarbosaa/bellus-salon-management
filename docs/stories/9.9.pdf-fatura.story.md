**Status:** InProgress

## Story

**As a** proprietário de salão,
**I want** gerar faturas em PDF profissional com todos os dados fiscais e QR Verifactu,
**so that** possa entregar ao cliente um documento fiscal válido e apresentável.

---

## Acceptance Criteria

1. Existe componente/função `generateFacturaPDF(facturaId)` que gera PDF completo
2. O PDF inclui: dados do emissor (NIF, nombre fiscal, endereço), dados do cliente (se disponível), número da fatura, data de emissão, linhas de serviço (descrição, quantidade, preço unitário, IVA, subtotal), resumo fiscal (base imponible, IVA, IRPF, total), forma de pagamento, QR Code Verifactu
3. O layout é profissional com a marca Bellus (logo, cores do salão)
4. O PDF usa a lib `jspdf` já instalada no projeto
5. O QR Code é renderizado como imagem no PDF (canto inferior direito)
6. O PDF é gerado client-side (para evitar carga no server) com dados buscados via server action
7. Botão "Download PDF" disponível na listagem de faturas e no detalhe
8. O nome do arquivo segue o padrão: `factura-{serie}-{numero}-{NIF}.pdf`
9. `npm run build` sem erros
