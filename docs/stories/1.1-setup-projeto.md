## User Story

**Como** desenvolvedor,
**Quero** ter o projeto Next.js configurado com Supabase e TypeScript,
**Para que** a equipe possa começar a implementar funcionalidades sobre uma base sólida.

Este é o ponto de partida do Bellus. O projeto já tem estrutura inicial no repositório (`bellus/` folder com Next.js), mas precisa ser configurado corretamente com todas as dependências e variáveis de ambiente.
