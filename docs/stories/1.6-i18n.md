## User Story

**Como** proprietário do salão,
**Quero** usar o sistema no meu idioma preferido (PT, ES, EN ou RU),
**Para que** a plataforma seja acessível para qualquer profissional em Valencia.

- [x] AC-01: Sistema detecta idioma do browser automaticamente (fallback ES)
- [x] AC-02: Seletor de idioma visível no header do dashboard
- [x] AC-03: Idioma selecionado persiste entre sessões (cookie com 1 ano)
- [x] AC-04: Conteúdo base do dashboard traduzido em PT, ES, EN, RU
- [x] AC-05: Datas e números formatados conforme locale do idioma
