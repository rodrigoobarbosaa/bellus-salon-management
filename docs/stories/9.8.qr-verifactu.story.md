# Story 9.8: QR Code Verifactu em Cada Fatura

**Status:** InProgress
**Epic:** EPIC-09 Verifactu
**Criado por:** River (@sm)
**Data:** 2026-03-30
**Depende de:** Story 9.4 (hash)
**Complexidade:** S (3 SP)

---

## Story

**As a** cliente do salão,
**I want** que cada fatura tenha um QR Code com dados verificáveis,
**so that** eu (ou a AEAT) possa verificar a autenticidade da fatura escaneando o QR.

---

## Acceptance Criteria

1. Existe módulo `src/lib/verifactu/qr-generator.ts` que gera QR data string conforme especificação Verifactu
2. O QR contém: URL de verificação AEAT + NIF emissor + número fatura + fecha emision + importe total + hash (primeiros 8 caracteres)
3. O formato do QR segue o padrão da AEAT (URL base: `https://www2.agenciatributaria.gob.es/wlpl/TIKE-CONT/ValidarQR?...`)
4. O campo `qr_data` da fatura é preenchido automaticamente no createFactura
5. A lib `qrcode` (já instalada no projeto) é usada para gerar a imagem QR
6. Existe componente React `<QRCodeVerifactu facturaId={id} />` que renderiza o QR
7. `npm run build` sem erros
